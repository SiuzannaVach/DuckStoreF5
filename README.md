### Instalación
1. Clona el repositorio:
   ```bash
   git clone https://github.com/SiuzannaVach/DuckStoreF5.git
   ```

2. Accede al directorio del proyecto:

```bash
cd DuckStoreF5
```

3. Instala las dependencias:

```bash
npm install
```

4. Inicia el entorno de desarrollo:

```bash
npm run dev
```
